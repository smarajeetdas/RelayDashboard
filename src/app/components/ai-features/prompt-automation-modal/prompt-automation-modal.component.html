Prompt Based Web Automation

Context

Test automation often requires extensive scripting, which is time-consuming and inflexible. Teams struggle to adapt test cases quickly as application flows evolve, resulting in slower test cycles and reduced test coverage.

Proposal

Relay's Prompt Based Web Automation enables test execution using simple natural language instructions:

  • • Translates prompts into executable test flows
  • • Requires no coding or scripting effort
  • • Adapts dynamically to UI and content changes
  • • Supports rapid authoring and maintenance of test cases

Outcome

  • • Faster test creation with minimal technical skills
  • • Increased test coverage across user journeys
  • • Quick adaptation to UI updates
  • • Empowered QA and business teams alike

Business Value

  • • 70% reduction in test authoring time
  • • Lower dependency on technical resources
  • • Faster go-to-market through accelerated test cycles
  • • Enhanced agility in response to product changes
