Prompt Based Web Automation

Context

Traditional test automation requires significant programming knowledge and maintenance. Codeless tools exist but have limitations in handling complex scenarios.

Proposal

Relay introduces prompt-based automation:

  • • Write automation instructions in plain English
  • • AI interprets and generates executable code
  • • Self-healing capabilities adapt to UI changes
  • • Handles complex scenarios like dynamic data and conditionals

Outcome

  • • Anyone can write test automation
  • • 80% faster test creation
  • • 60% reduction in maintenance effort
  • • Tests that adapt to application changes

Business Value

  • • Democratized test automation
  • • Reduced dependency on specialized engineers
  • • Faster time-to-market with increased test coverage
  • • Lower total cost of ownership for test automation
